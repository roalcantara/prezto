Yarn
===

Enhances the [Yarn][1] by providing a few aliases.

### Aliases

-   `y` runs `yarn`
-   `ya` runs `yarn add`
-   `yb` runs `yarn build`
-   `yl` runs `yarn lint`
-   `yt` runs `yarn test`
-   `ys` runs `yarn start`
